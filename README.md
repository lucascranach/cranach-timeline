# Lucas Cranach Timeline

## Installation

1. Das Repository klonen

```bash
git clone git@github.com:lucascranach/cranach-timeline.git
```

2. In das Verzeichnis wechseln

```bash
cd cranach-timeline
```

3. Abhängigkeiten installieren

```bash
npm i
```

4. Entwicklungsserver starten

```bash
npm run dev
```

5. Im Browser öffnen

Öffnen Sie [http://localhost:5172/timeline](http://localhost:5172/timeline) in Ihrem Webbrowser, um die Anwendung zu sehen.
